# TND

🧠 Outil de Dépistage TDAH / TSA (Non Diagnostique)

📌 Description
Cet outil propose un questionnaire d’auto-évaluation visant à identifier des traits compatibles avec un TDAH (Trouble du Déficit de l’Attention avec ou sans Hyperactivité) et/ou un TSA (Trouble du Spectre de l’Autisme).

⚠️ Il s’agit d’un outil de dépistage uniquement, conçu pour :
- sensibiliser
- orienter la réflexion
- encourager une démarche d’exploration personnelle ou professionnelle

🎯 Objectifs
- Fournir une première grille de lecture accessible
- Aider à repérer certains profils neuroatypiques
- Encourager une orientation vers des professionnels qualifiés si nécessaire
- Proposer une approche nuancée (pas de catégorisation brutale)

⚙️ Fonctionnement
- Questionnaire structuré (inspiré de la littérature scientifique)
- Scores par dimensions (attention, impulsivité, cognition sociale, rigidité, etc.)
- Résultat interprété sous forme de tendances, et non de diagnostic

🧩 Dimensions évaluées
- Attention / distractibilité
- Impulsivité / hyperactivité
- Flexibilité cognitive
- Interaction sociale
- Communication
- Sensibilité sensorielle
- Centres d’intérêt spécifiques

⚠️ Avertissement important

❗ Cet outil n’est pas un dispositif médical.
- Il ne permet pas de poser un diagnostic
- Il ne remplace pas une évaluation réalisée par un professionnel de santé
- Les résultats doivent être interprétés avec prudence
- Toute inquiétude doit mener à une consultation (psychologue, neuropsychologue, psychiatre, médecin)

🧪 Limites
- Basé sur de l’auto-déclaratif (biais possibles)
- Chevauchement entre :
- - TDAH
- - TSA
- - anxiété
- - trauma
- - hypersensibilité
- - Risque de faux positifs / faux négatifs

🧠 Positionnement
Cet outil s’inscrit dans une logique de :
- compréhension des fonctionnements
- approche non stigmatisante
- respect de la singularité de chaque individu

Il vise à éviter les conclusions simplistes du type :
“Vous êtes TDAH” ou “Vous êtes autiste”

📊 Interprétation des résultats
Les scores indiquent des tendances :
- Faible : peu d’éléments significatifs
- Modéré : présence de certains traits
- Élevé : profil à explorer plus en profondeur

👉 Un score élevé n’est pas un diagnostic.
